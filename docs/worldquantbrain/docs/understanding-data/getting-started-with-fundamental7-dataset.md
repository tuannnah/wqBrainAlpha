**Getting Started with fundamental7 dataset (Comprehensive Fundamentals Data)**

The fundamental7 dataset provides a comprehensive collection of company financial metrics extracted directly from regulatory filings, offering deep insights into corporate financial performance, operational efficiency, and cash management across time periods. This extensive dataset integrates standard financial statement elements with supplementary footnote information, allowing users to identify and clarify anomalies in primary financial statements that might otherwise go undetected.

**The Dataset provides:**

* Earnings Performance Metrics: Multiple EPS calculations (basic, diluted, with/without extraordinary items), income components, and earnings quality indicators
* Financial Position Indicators: Balance sheet components, liquidity measures, debt structure, and capital allocation metrics
* Cash Flow Dynamics: Detailed cash flow statement items tracking operating, investing, and financing activities
* Capital Structure Elements: Equity composition, debt ratios, and shareholder-focused metrics like dividends and share count
* Operational Efficiency Measures: Cost structure, margin profiles, and resource utilization metrics
