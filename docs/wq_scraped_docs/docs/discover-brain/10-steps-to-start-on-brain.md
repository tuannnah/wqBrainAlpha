**10 Steps to Start on BRAIN**

1. Watch the [“Introduction to Alphas”](https://platform.worldquantbrain.com/learn/courses/introduction-alphas) training series
2. Read the [starter pack series](https://platform.worldquantbrain.com/learn/documentation#:~:text=Documentation-,Discover%20BRAIN,-4%20Articles) under “Discover BRAIN” section
3. Simulate three simple formulas using the data field: close and the operator: rank
4. Understand the simulation settings, starting with [delay and neutralization](https://platform.worldquantbrain.com/learn/documentation/create-alphas/simulation-settings) .
5. Click the Example button on the bottom left of the [Simulate](https://platform.worldquantbrain.com/simulate) page for examples. Try to improve these expressions using the Hint and then submit
6. Understand the [Simulation Results](https://platform.worldquantbrain.com/learn/documentation/interpret-results/parameters-simulation-results) and submission tests.
7. Simulate Alphas discussed in the "Create Alphas" sections [Run your First Alpha](https://platform.worldquantbrain.com/learn/documentation/create-alphas/running-your-first-alpha) and [Another Sample Alpha](https://platform.worldquantbrain.com/learn/documentation/create-alphas/another-sample-alpha). Pay attention to the settings.
8. Check the [Events](https://platform.worldquantbrain.com/events) page to attend training webinars.
9. Read the [BRAIN TIPS series](https://support.worldquantbrain.com/hc/en-us/search?filter_by=community&query=%5Bbrain+tips&utf8=%E2%9C%93) and the [Beginner](https://support.worldquantbrain.com/hc/en-us/search?content_tags=01GVTJQR8518KQMHP1JMCZ4Y99&utf8=%E2%9C%93) posts in FAQs/ Forums
10. Try to create Alphas using available [Price volume](https://platform.worldquantbrain.com/data/data-sets/pv1) data fields or and, [Operators](https://platform.worldquantbrain.com/learn/data-and-operators/operators)
